# 📘 PHP Workshop – Atelier2

This repository contains solutions for a PHP workshop about **variables, constants, and data types**.

It includes the following exercises:

- ✅ Exercise 1: Declaration and display of variables  
- ✅ Exercise 2: Script correction with errors  
- ✅ Exercise 4: Dynamic sentence with variables  
- ✅ Exercise 5: Conversion and concatenation  
- ✅ Exercise 6: Data type verification  


Each file corresponds to one exercise from the workshop.

---

## 🧪 Exercises Description

### 🔹 Exercise 1: Variables Declaration
- Declare variables (string, int, float, bool)
- Display them using `echo`
- Check their types using `var_dump()`

---

### 🔹 Exercise 2: Script Correction
- Correct syntax errors in a PHP script
- Add a constant
- Explain corrections using comments

---

### 🔹 Exercise 4: Dynamic Sentence
- Use variables and constants to create a sentence
- Modify a variable and observe the result

---

### 🔹 Exercise 5: Conversion & Concatenation
- Add a string and an integer
- Test the difference between `+` and `.` operator
- Understand automatic type conversion in PHP

---

### 🔹 Exercise 6: Data Type Verification
- Create an associative array
- Display values and types with `var_dump()`
- Test functions like `is_string()`, `is_int()`, etc.
